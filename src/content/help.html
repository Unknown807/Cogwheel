Introduction

Welcome to Cogwheel a simple cognitive calculator based roughly on KLM-GOMS (Keystroke Level Model -
Goals Operators Methods Selection Rules), with the design made similar to an existing program called
Cogulator.

The purpose of the program is to calculate how long a series of tasks when done by an expert user would
take and how much the user would have to rememeber as well (e.g mental maths). Normally in KLM-
GOMS users making errors is not taken into account, but Cogwheel has the Forget keyword to account
for things like memory loss. So some consideration has been made for errors.

How It Works

The way the total time for a series of tasks to complete is calculated is by counting up all the
seconds and milliseconds written in the editor and then displaying it above next to the clock icon.
The program also watches out for <2> like this in brackets as a multiplier for a task, so in this
case the 2 with an action taking 100 milliseconds would be 200 milliseconds. In case the time is not
mentioned next to the task, the program uses the default times that you either changed in the
settings file or if the settings file doesn't exist it uses hardcoded default values.

Chunks are made by individuals to better memorise something, chunking is a phenomenon whereby an
individual groups responses when performing a memory task. For example a phone number like 123456789
when chunked can become 123-456-789.

So the way total chunks are calculated is by counting the maximum amount of chunks a user has to store
at any one point throughout the defined series of tasks and the use of the Forget keyword will reset the
counter to see if any tasks later will require the user to store more information than before.

Files

By default the editor is empty so you can start typing immediately, if not you can always create a new
file (which will overwrite the editor). You can also open saved files from anywhere on your computer,
even though your files will be saved in the program directory most of the time. When you do want to
save a file, just press the save file button on the toolbar. For a successful save make sure to name
your file appropriately with letters and numbers, otherwise you will get an error popup. You will know
if you saved your file when the save button lights up in green for a second.

Settings

The settings page allows you to change the default time it takes to complete certain tasks in seconds.
So if you know a Click will almost always take 200ms for you, you would write in the text field for
the Click keyword '0.2'. At the bottom of the page you also have a number of themes you can select for
the program.

When you're finished making changes you can either click the back button which will take you to editor
page and will automatically save your changed settings or you can always click the 'save settings'
button manually.
